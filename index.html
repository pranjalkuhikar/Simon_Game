Simon Game

Press any key to start the game

How to Play


  1. The goal of the Simon game is to remember the pattern of lights and get the highest score.
  2. To Start the game press any key.
  3. At the start of the game, one of the buttons lights up. The player must press that light up button.
  4. Then the game light up another button but you need to press 2 button on the second level. ie. Pevious one and new one.
  5. The player must reproduce the chain of lights using their memory, As the game progresses.
  6. If the player is fail then display the score by flashing red lights when the player doesn't complete a sequence successfully.
